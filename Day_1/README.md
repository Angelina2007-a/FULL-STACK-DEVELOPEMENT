# 🥛 Nestlé Product Showcase Website

A simple product showcase website built using **HTML5** and **CSS3** as part of my **Full Stack Development Journey – Day 1**.

## 📌 About

This project is a static landing page inspired by Nestlé products. It showcases different product categories using a clean layout, CSS styling, and interactive hover effects.

## ✨ Features

- Gradient background
- Hero banner
- Dairy Products section
- Grocery Products section
- Packaged Foods section
- Product cards with hover animations
- "Order Now" buttons
- Flexbox-based layout
- Simple and clean UI

## 🛠️ Tech Stack

- HTML5
- CSS3
