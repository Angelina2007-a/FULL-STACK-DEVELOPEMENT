# 🦸 Avengers Assemble Landing Page

A simple **Avengers-themed landing page** built using **HTML** and **CSS**. The project features a cinematic cover image, an Avengers-themed title, and a responsive character gallery with hover effects.

## ✨ Features

- 🖼️ Full-width Avengers cover image
- 🎨 Marvel-inspired dark gradient background
- 🔤 Avengers title using **Bebas Neue** Google Font
- 🦸 Character gallery displayed using **CSS Grid**
- ✨ Smooth hover zoom animation on images
- 🌈 Colored shadow effects for each character card
- 📱 Clean and responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Bebas Neue)
